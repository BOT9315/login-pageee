# 🔐 Ludiflex — Login & Registration Page

A sleek, glassmorphism-styled login and registration UI built with pure HTML, CSS, and JavaScript. Features smooth slide transitions between login and sign-up forms, a responsive navbar, and a frosted-glass aesthetic over a full-screen background image.

---

## 📸 Preview

> The page uses a local background image (`images/image.jpeg`). Add your own image to that path to see the full effect.

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/BOT9315/login-pageee.git
cd login-pageee
```

### 2. Add a background image

Place any JPEG image at:

```
images/image.jpeg
```

### 3. Open in browser

Simply open `index.html` in your browser — no build tools or server required.

```bash
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

## ✨ Features

- 🔄 **Smooth form transitions** — Login and Register panels slide in/out with CSS transitions
- 🌫️ **Glassmorphism design** — Semi-transparent frosted inputs and buttons
- 📱 **Fully responsive** — Hamburger menu on mobile (≤786px), stacked form on small screens (≤540px)
- 🎨 **No dependencies** — Pure HTML, CSS, and vanilla JavaScript
- 🔡 **Poppins font** + **Boxicons** loaded via CDN

---

## 🛠️ Built With

| Technology | Purpose |
|------------|---------|
| HTML5 | Page structure and forms |
| CSS3 | Styling, transitions, responsive layout |
| Vanilla JavaScript | Form switching & mobile menu toggle |
| [Poppins (Google Fonts)](https://fonts.google.com/specimen/Poppins) | Typography |
| [Boxicons](https://boxicons.com/) | Input field icons |

---

## 📱 Responsive Breakpoints

| Breakpoint | Behavior |
|------------|----------|
| > 786px | Full navbar with Sign In / Sign Up buttons |
| ≤ 786px | Hamburger menu, nav buttons hidden |
| ≤ 540px | Full-width form, stacked first/last name fields |
